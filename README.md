# Setup 

1. Clone repository

```bash
git clone https://github.com/Sele3/assessment1.git
cd assessment1
```

2. Install dependencies

```bash
npm install
```

3. Start the server

```bash
npm start
```

# Technologies Used

- **ReactJS** as the frontend framework
- **TailwindCSS** for styling
- **Axios** for making API requests
